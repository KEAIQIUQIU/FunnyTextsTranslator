# 多轮翻译工具

## 项目简介

这是一个通过多次随机语言中转实现翻译文本的工具，旨在通过多轮翻译过程获得有趣的结果。工具使用本地运行的LibreTranslate服务，支持多种语言互译，包含直观的图形界面和翻译报告生成功能。

## 主要功能

- **多轮随机翻译**：将文本通过随机选择的多国语言中转后翻译成目标语言
- **翻译过程可视化**：实时显示每轮翻译结果和当前语言路径
- **翻译报告生成**：自动记录完整翻译路径和中间结果
- **本地翻译服务**：基于LibreTranslate引擎，保护隐私无需联网

## 安装与使用

### 前置要求
- Python 3.6+
- LibreTranslate

### 安装步骤
1. 安装Python依赖：
```bash
pip install -r requirements.txt
```

2. 启动LibreTranslate服务：
```bash
libretranslate --host 127.0.0.1 --port 5000
```

### 使用说明
1. 确保LibreTranslate服务在5000端口运行
2. 运行程序：`python GUI.py`
3. 输入要翻译的文本
4. 设置目标语言和循环次数
5. 点击"开始翻译"按钮
6. 查看实时翻译过程和最终结果

## 技术实现

### 组件
- `GUI.py` - 图形用户界面
- `translate_core.py` - 多轮翻译核心逻辑

## 注意事项

1. 确保LibreTranslate服务在5000端口运行
2. 循环次数建议在5-20次之间
3. 报告默认保存在`./output/`目录
4. 支持LibreTranslate的所有可用语言
